# 여행상품 웹 서비스 프로젝트

제품들을 조회하고, 장바구니에 추가하고, 또 주문을 할 수 있는 여행상품 웹 서비스 제작 프로젝트입니다. <br />
**웹 구현 예시** (링크는 프로젝트 기간에만 유효합니다)

배포한 사이트 : https://yeonol.herokuapp.com/   

<br>

## 주요 사용 기술

### 1. 프론트엔드

- **Vanilla javascript**, html, css 
- Font-awesome

### 2. 백엔드

- **Express** (nodemon, babel-node로 실행됩니다.)
- Mongodb, Mongoose
- cors
- 이외
